# E-Learning System

An online E-Learning web application developed using ASP.NET MVC and SQL Server.  
This project helps students learn courses online and allows admins/mentors to manage educational content efficiently.

---

## 📌 Project Overview

The E-Learning System is designed to provide an online learning platform where users can:

- Register and login securely
- Browse available courses
- Enroll in courses
- Watch video lectures
- Download study materials
- Track learning progress
- Manage users and courses through admin panel

## 🏗️ Architecture

The project follows the MVC (Model-View-Controller) architecture.

### Model
Handles application data and business logic.

### View
Displays user interface to users.

### Controller
Handles user requests and application flow.
